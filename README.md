## Hybrid DESTINY Workflow

Enables sensitivity analysis and gradient based optimization of memory technology configurations for a given architecture configuration generated by c++ version of destiny. 

### 1. Create and activate environment
```
source env_start.sh
```

### 3. Run the hybrid workflow
```
cd src
python hybrid_destiny_workflow.py \
  --config destiny_3d_cache/config/sample_SRAM_2layer.cfg \
  [--output <output_file_name>] \
  [--python-script symbolic_access_time_FIXED.py] \
  [--skip-cpp] \
  [--symbolic-tech-model]
```
